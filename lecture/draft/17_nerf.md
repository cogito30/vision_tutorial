# Step 4. 2D 사진을 3D 공간으로: 3D Gaussian Splatting과 Nerfstudio 마스터하기
안녕하세요! 컴퓨터 비전 심화 튜토리얼 네 번째 시간입니다.

지금까지 우리는 이미지 안에서 객체를 탐지하고(YOLO, RT-DETR), 형태를 정교하게 분할하는(SAM 2) 기술을 배웠습니다. 하지만 이 기술들은 모두 '2D 평면'에 머물러 있었습니다.
자율주행 자동차, 로봇 공학, 애플 비전 프로(Vision Pro)와 같은 공간 컴퓨팅(Spatial Computing) 시대에는 평면을 넘어 **공간 전체(3D)를 이해하고 재구성**하는 기술이 필수적입니다.

오늘은 최근 1~2년간 3D 비전 업계를 완벽하게 평정해 버린 혁명적인 기술, **3D Gaussian Splatting (3DGS)** 에 대해 알아보고, 내 방을 직접 3D로 만들어보는 실습을 진행하겠습니다! 🚀

## 1. 3D 비전의 혁명: NeRF에서 3DGS로
2D 사진 여러 장을 바탕으로 3D 공간을 만들어내는 기술은 꾸준히 발전해 왔습니다.

- **NeRF (Neural Radiance Fields, 2020년)**: 여러 각도에서 찍은 사진을 인공지능(딥러닝)에게 보여주면, AI가 빛의 방향과 색상을 추론하여 안 찍은 각도의 모습까지 렌더링해 주는 마법 같은 기술이었습니다.
하지만 치명적인 단점이 있었습니다. 공간을 학습하는 데 수~수십 시간이 걸리고, 렌더링 속도가 5 FPS 정도로 너무 느려서 실시간 서비스(게임, 웹 뷰어)에 사용할 수 없었습니다.
- 🏆 **3D Gaussian Splatting (3DGS, 2023년 말)**:
NeRF의 속도 문제를 완벽히 해결한 기술입니다. 무거운 딥러닝 네트워크(MLP)를 걷어내고, 3D 공간을 수백만 개의 **'반투명한 타원형 물방울(Gaussian)'** 들로 가득 채우는 방식을 택했습니다. 이 물방울들을 캔버스에 물감을 뿌리듯(Splatting) 고속으로 투사합니다.
결과는 충격적이었습니다. 학습 시간은 몇 분으로 단축되었고, 렌더링 속도는 초당 **100 FPS** 이상의 고화질 실시간 뷰잉이 가능해졌습니다. 현재 모든 산업계의 디지털 트윈, 3D 캡처 앱(Polycam, Luma 등)이 이 3DGS 기술로 넘어가고 있습니다.

## 2. 실습 준비: Nerfstudio 설치하기
3DGS를 밑바닥부터 파이썬으로 구현하기는 매우 복잡합니다. 다행히도 UC 버클리 연구진이 만든 3D 비전 오픈소스 프레임워크인 **Nerfstudio**를 사용하면 명령어 몇 줄로 강력한 3DGS 파이프라인을 구축할 수 있습니다.

터미널을 열고 가상환경(`cv_env`)에 아래 명령어들을 순서대로 입력하여 설치합니다.
(주의: 3D 렌더링 패키지는 설치가 꽤 무거울 수 있습니다.)
```
# 1. 3D 공간의 카메라 위치(Pose)를 추정해 주는 COLMAP 도구 설치 (Mac용)
brew install colmap

# 2. Nerfstudio 설치 (PyTorch 기반)
pip install nerfstudio
```
(💡 Mac Apple Silicon(M1/M2 등) 환경에서도 `nerfstudio` 구동이 지원됩니다. 단, 3DGS 학습은 GPU 연산량이 방대하여 Mac에서는 시간이 다소 걸릴 수 있습니다. 가장 빠른 속도를 원하신다면 구글 Colab이나 AWS의 NVIDIA GPU 서버를 활용하는 것도 좋은 방법입니다.)

## 3. 실습: 내 방을 3D 디지털 트윈으로 만들기

#### Step 1. 스마트폰으로 동영상 촬영하기
스마트폰을 들고 3D로 만들고 싶은 공간(내 방, 책상 위, 귀여운 피규어 등)을 동영상으로 촬영합니다.

- **촬영 팁**: 대상을 가운데 두고 천천히 원을 그리며(Orbit) 빙 둘러서 찍습니다. 너무 빠르게 움직이거나 흔들리면 안 됩니다. (약 15~30초 분량)
- 촬영한 동영상을 컴퓨터로 옮기고 이름을 `my_room.mp4`로 변경합니다.

#### Step 2. 데이터 전처리 (카메라 위치 추정)
동영상에서 프레임(이미지)을 추출하고, COLMAP을 이용해 "이 사진이 어느 위치(x, y, z 좌표)에서 찍혔는지"를 계산하는 과정입니다. 터미널에 아래 명령어를 입력하세요.
```
# 동영상을 기반으로 학습용 데이터셋 폴더 생성
ns-process-data video --data my_room.mp4 --output-dir data/my_room
```

이 작업이 끝나면 `data/my_room` 폴더 안에 여러 장의 사진과 `transforms.json`(카메라 위치 정보) 파일이 생성됩니다.

#### Step 3. 3DGS (Splatfacto) 모델 학습!
이제 대망의 3D 공간 학습입니다. Nerfstudio에서는 3DGS 기반의 모델을 `splatfacto`라고 부릅니다.
```
# 3DGS 모델 학습 시작!
ns-train splatfacto --data data/my_room
```

명령어를 실행하면 터미널에 실시간 뷰어 접속 주소(예: `https://viewer.nerf.studio/...`)가 뜹니다. 크롬 브라우저로 해당 주소에 접속하면, 허공에 점들이 흩뿌려진 상태에서 실시간으로 내 방이 점점 선명하게 3D 물방울들로 채워져 완성되는 과정을 눈으로 직접 볼 수 있습니다!

#### Step 4. 실시간 웹 뷰어로 감상하기
학습이 100% 완료되면(보통 몇천 번의 반복 스텝 이후), 생성된 설정 파일(config.yml)을 로드하여 언제든 3D 공간을 자유롭게 날아다니며 감상할 수 있습니다.
```
# 경로의 날짜/시간 부분은 본인의 outputs 폴더 안을 확인하여 수정하세요.
ns-viewer --load-config outputs/my_room/splatfacto/2026-xx-xx_xxxxxx/config.yml
```

이제 여러분은 마우스를 드래그하여 현실 공간을 완벽하게 복제한 3D 가상 공간을 마음껏 둘러볼 수 있습니다.
